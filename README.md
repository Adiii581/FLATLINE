# FLATLINE

FLATLINE is a retro/arcade style medical simulation game where players use tests and diagnoses to solve infinite, AI-generated patient cases. Powered by the Google Gemini API and a robust FastAPI backend, it combines high-stakes medical deduction with a nostalgic 80s terminal aesthetic.

---

## Tech Stack

### Frontend
* React (Vite)
* Tailwind CSS

### Backend
* Python
* FastAPI
* Pydantic

### AI Engine
* Google Gemini API (Model: gemini-2.5-flash)

---

## Prerequisites

Before running the game, ensure you have the following installed:

* Node.js (v16 or higher)
* Python (v3.9 or higher)
* A Google Gemini API Key

---
